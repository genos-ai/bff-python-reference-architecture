# Plans

Implementation plans, feature proposals, and progress tracking.

## Naming Convention

```
001-plan-feature-name.md
002-plan-migration-topic.md
```

## Adding a Plan

Create a new file with the next available number. Each plan should include:

- **Goal** — what we're building and why
- **Scope** — what's in and out of scope
- **Tasks** — numbered checklist of work items
- **Dependencies** — what must exist before starting
- **Acceptance Criteria** — how we know it's done
